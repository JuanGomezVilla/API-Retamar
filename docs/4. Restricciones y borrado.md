# Restricciones y borrado
La base de datos cuenta con restricciones. Las tablas están relacionadas entre sí por claves foráneas. Por ejemplo, una asignatura la imparte un profesor, y este dato está relacionado con la tabla de profesores. Por lo tanto, si un profesor _fuera despedido_ dicho dato no se borrará, sin embargo, cabe la posibilidad de indicar que ya no continúa trabajando. Existe una función para vaciar las tablas de forma apropiada, aunque todavía no está implementada.
